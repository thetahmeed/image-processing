## Create and activate the env

```bash
python3 -m venv .venv
```

```bash
source .venv/bin/activate
```

## Install all dependencies

```bash
pip install -r requirments.txt
```

### Run

```bash
python3 processing_by_cv2.py
```

```bash
python3 processing_by_pillow.py
```

```bash
python3 equalization.py
```
